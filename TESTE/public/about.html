About the project

We, Pedro Ferreira and João Figueiredo, are students at IPT (Instituto Politécnico de Tomar) and developed this application as part of the IoT course. Our student numbers are:

  • Pedro Ferreira: 21264
  • João Figueiredo: 24179

This application is designed to provide real-time data visualization and monitoring for various sensors. It displays moisture levels, weather information, and humidity data over time.

Our goal with this project was to create a user-friendly and visually appealing interface for accessing and analyzing sensor data. We hope you find this application useful and informative.
